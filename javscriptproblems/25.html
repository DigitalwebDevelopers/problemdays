Day 25 JavaScript Challenge — “Rearrange for Palindrome?”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a string, determine whether its characters can be rearranged to form a palindrome. Ignore spaces and consider case-insensitive comparison.

🧪 Example Input:

const input = "civic"; // true  
const input = "ivicc"; // true  
const input = "hello"; // false

🔸 Your Task to Perform

Count the frequency of each character. For a string to form a palindrome, at most one character can have an odd frequency.

Return `true` if it’s possible to rearrange into a palindrome, else `false`.

🧪 Example Output:

true // because “ivicc” can be rearranged to “civic”

🔸 Bonus Tip for You

Try solving it without sorting. Can be done using a simple object or Map.

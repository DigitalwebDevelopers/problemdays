Day 90 JavaScript Challenge — “Generate n-bit Gray Code”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

In digital logic, a Gray code is a binary numeral system where two successive values differ in only one bit.

Your task is to write a function that generates a Gray code sequence for a given number of bits (n). Return an array of integers representing the sequence.

🧪 Sample Input:

grayCode(2)

🧪 Output:

[0, 1, 3, 2]
// Binary: 00 → 01 → 11 → 10 (only one bit changes each step)

🔸 Core Logic:

  • Use a loop from 0 to (2^n - 1), and at each step generate: i ^ (i >> 1)
  • XORing a number with itself shifted right by 1 produces the Gray code.

🎯 Bonus Insight:

  • Gray codes are heavily used in Karnaugh maps, digital encoders, and to prevent signal misinterpretation in circuit design.
  • They reduce errors by ensuring only one bit flips at a time, perfect for hardware-based state transitions.

Day 99 JavaScript Challenge — “Nesting Depth of Parentheses”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given a string that contains only parentheses. Your task is to calculate the maximum depth of nested parentheses.

The depth increases with every opening bracket and decreases with every closing one. You need to determine how deep the nesting goes.

🧪 Sample Input:


const expression = "(1+(2*3)+((8)/4))+1";

// Output:
3

💡 Explanation:

  • The deepest nested parentheses in the expression is ((8)/4) which is at depth 3.

🔸 Real-World Use Case:

  • Such logic is useful in parsing mathematical expressions, validating syntax in compilers, and ensuring correct nesting in HTML/XML/JSON documents.

Day 53 JavaScript Challenge — “Sum of Differences”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given an array of numbers. Your task is to calculate the sum of the differences between each pair of adjacent elements after sorting the array in descending order.

🧪 Example Input: 

const input = [2, 1, 10];

🔸 Your Task to Perform

First, sort the array in descending order. Then, calculate the difference between each adjacent pair and return the total of all these differences.

  • If array is empty or has one element, return 0.
  • Focus on clean code using sorting + loops.

🧪 Expected Output: 

9 // Explanation: [10, 2, 1] → (10-2) + (2-1) = 8 + 1 = 9

🔸 Bonus Tip for You

Use arr.sort() with a custom compare function to sort in descending order. Try writing the entire solution using only one loop after sort.

Be cautious of edge cases like empty or single-element arrays. Always test those!

Day 42 JavaScript Challenge — “Group Scores by Subjects”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given an array of student score objects. Your task is to group scores by subject and return an object where each key is a subject, and the value is the total score of all students in that subject.

🧪 Example Input: 

const scores = [
  { name: "Amit", subject: "Math", score: 90 },
  { name: "Neha", subject: "Science", score: 80 },
  { name: "Rohit", subject: "Math", score: 85 },
  { name: "Neha", subject: "Math", score: 70 },
  { name: "Ravi", subject: "Science", score: 95 },
];

🔸 Your Task to Perform

  • Loop through the array
  • Group scores by subject
  • Return an object like { Math: 245, Science: 175 }

🧪 Expected Output: 

{
  Math: 245,
  Science: 175
}

🔸 Bonus Tip for You

Make sure to use pure loops like for...of or forEach — and try to avoid using .reduce if you're still early in JS learning.

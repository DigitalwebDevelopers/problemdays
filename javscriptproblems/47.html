Day 47 JavaScript Challenge — “Bus Stop Waiting Time Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re working with a city transport app. Buses run at scheduled intervals. You receive a list of bus stop arrival times in minutes (past the hour), and a user’s arrival time.

Your task is to figure out how long they have to wait for the next bus. If no bus is available, show a message to "Take a cab".

🧪 Example Input:

const busTimings = [5, 15, 25, 40, 55];
const userTime = 26;

🔸 Your Task to Perform

  • Loop through the bus timing array and find the next bus that arrives after the user's time.
  • Calculate and print how many minutes they need to wait.
  • If no bus is available, print "Take a cab".

🧪 Example Output:

Next bus in: 14 minutes

🔸 Bonus Tip for You

Don't sort the array. Assume it's already sorted.

Edge Case: If user's time is more than all bus timings — show "Take a cab".

Day 65 JavaScript Challenge — “Check if Orders Are Balanced”

Taught By Code with Deepanshu Knox

🛒 Problem Of the Day —

An e-commerce platform tracks user orders as a list of transaction amounts. Some are purchases (positive numbers), and some are refunds (negative numbers).

Your job is to determine if the user's account is "balanced" — meaning that the total purchases equal total refunds.

🧪 Example Input:

const transactions = [100, -50, 20, -20, -50];

🔸 Your Task to Perform

  • Calculate the total sum of transactions.
  • Return true if the sum is exactly 0 (balanced), otherwise return false.

🧪 Expected Output:

true

🔸 Bonus Tip for You

  • Use Array.reduce() to calculate the net transaction value in one line.
  • This type of logic can help in ledger balancing, financial audit tools, and refund validation systems.

Day 6 JavaScript Challenge — “Find Highest Scorer”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given an object where keys are student names and values are their scores. Your task is to find the name of the student who has scored the highest.

🧪 Example Input: 


const scores = {
  Arjun: 88, Simran: 91, Riya: 77, 
  Kabir: 95, Tara: 89
};

🔸 Your Task to Perform

Loop through the object using a `for...in` loop.

Track the highest score and its corresponding student name.

Return the name of the highest scorer.

🧪 Example Output: 

"Kabir" // because he scored 95, the highest

🔸 Bonus Tip for You

Don't use `Object.entries()` or `.sort()` — stick to basic loops.

Try to handle empty object edge case too (return `"No data"`).

Day 51 JavaScript Challenge — “Broken Keyboard”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

A user is trying to type a message, but their keyboard is broken. Some of the keys are non-functional (given in an array). You are given two strings:

  • `message`: The intended message the user wants to type.
  • `brokenKeys`: An array of characters representing keys that do not work.

Your task is to check whether the user can successfully type the message using the remaining working keys.

🧪 Example Input:

const message = "javascript";
const brokenKeys = ['a', 's', 'v'];

🔸 Your Task to Perform

  • Loop through the `message`
  • If any character from `message` exists in `brokenKeys`, return `"Message cannot be typed"`
  • Otherwise, return `"Message can be typed"`

🧪 Expected Output:

// Output: "Message cannot be typed"

🔸 Bonus Tip for You

  • Make it case-insensitive by converting both strings and array values to lower case.
  • Try to use `.includes()` smartly instead of nested loops.

Day 20 JavaScript Challenge — “Balanced Brackets Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given a string that contains only parentheses — round (), curly {}, and square []. Your task is to determine if the brackets are balanced or not.

A string is considered balanced when each opening bracket has a correct closing match in the proper order.

🧪 Example Input:

const str = "{[()]}";

🔸 Your Task to Perform

Use a stack-like approach. Push opening brackets. Pop and match when closing ones appear. If at the end stack is empty and no mismatches found — return true.

🧪 Example Output:

true // it's a valid sequence

🔸 Bonus Tip for You

Try mapping closing brackets to their opening ones using an object — it makes comparisons cleaner.

Handle edge cases like odd-length strings or unbalanced pairs like "[(])".

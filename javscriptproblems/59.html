Day 59 JavaScript Challenge — “Validate Email in Form”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are working on a user registration form. One of the key tasks is to validate the entered email to ensure it follows proper syntax before submitting to the server.

🧪 Example Input: 

const email = "john.doe@example.com";

🔸 Your Task to Perform

  • Validate the email using a regular expression.
  • Return true if the email is in a valid format, otherwise return false.
  • Avoid using third-party libraries.

🧪 Example Output: 

true // Email is validated

🔸 Bonus Tip for You

  • You can use this regex pattern: /^[\w.-]+@[a-zA-Z\d.-]+\.[a-zA-Z]{2,}$/
  • Test edge cases like missing “@”, invalid domain, or empty input.

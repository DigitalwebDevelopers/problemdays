Day 91 JavaScript Challenge — “Message Decoder from Cipher”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're building a feature for a secret messaging app. Messages are encoded using numbers where:
• 1 → A, 2 → B, ..., 26 → Z.

Your job is to write a function that returns the total number of ways a given numeric string can be decoded into letters.

🧪 Sample Inputs:

decodeWays("12");     // Output: 2   → "AB", "L"
decodeWays("226");    // Output: 3   → "BZ", "VF", "BBF"
decodeWays("06");     // Output: 0   → Invalid because no letter starts with 0

⚙️ Constraints:

  • Input will be a string of digits only (0-9).
  • Leading 0s without valid mapping make decoding invalid.

🧠 Bonus Insight:

  • This problem is a classic case of Dynamic Programming. You can solve it recursively by checking combinations of one and two digits.
  • Real-world applications include decoding OTPs, license keys, or language pattern predictions.

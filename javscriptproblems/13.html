Day 13 JavaScript Challenge — “Rearrange to Target”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given two arrays arr1 and arr2, return true if you can rearrange the elements of arr1 to get arr2. Order doesn't matter, but count and values should match.

🧪 Example Input:

const arr1 = [1, 2, 3, 4];
const arr2 = [4, 3, 2, 1];

🔸 Your Task to Perform

Use sorting or frequency counting to compare both arrays.

If both arrays have same numbers with same frequency, return true. Else return false.

Don't use JSON.stringify() or any serialization technique.

🧪 Example Output:

true

🧠 Bonus Tip for You

Instead of sorting, try using objects to count occurrences.

Handle edge cases like different lengths or different types (e.g., string vs number).

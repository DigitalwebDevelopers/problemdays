Day 19 JavaScript Challenge — “Sum Before Zero”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of numbers, calculate the sum of all elements before the first zero appears. If there's no zero, return the sum of the full array.

🧪 Example Input:

const nums = [3, 7, 1, 0, 9, 5];

🔸 Your Task to Perform

  • Start from beginning of array.
  • Keep adding numbers until you hit zero.
  • Stop immediately if zero is found.

🧪 Example Output:

11 // because 3 + 7 + 1 = 11 (stops before zero)

🔸 Bonus Tip for You

Use a loop and `break` when zero is found.

Try handling negative numbers or an empty array gracefully.

Day 67 JavaScript Challenge — “Track Response Gaps”

Taught By Code with Deepanshu Knox

🧾 Problem Of the Day —

A customer support system logs messages based on timestamps (in minutes). You are given an array of timestamps when customers sent queries.

Your task is to calculate the maximum gap between any two queries — i.e., find the largest time where no customer sent a query.

🧪 Example Input:

const timestamps = [5, 15, 25, 60, 63];

🎯 Output:

35 // This is the Track

🔸 Your Task to Perform

  • Sort the timestamps first.
  • Find the max difference between consecutive timestamps.

💡 Real World Use Case

  • Call centers can use this to identify idle periods and optimize staff shifts.

Day 60 JavaScript Challenge — “Invoice Data & Calculate Total”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are building a billing system. Users enter a list of purchased items — each with a price and quantity. Your task is to process this data and return the total bill amount (formatted as currency).

🧪 Example Input: 

const invoice = [
  { item: "Apple", quantity: 2, price: 30 },
  { item: "Milk", quantity: 1, price: 50 },
  { item: "Bread", quantity: 3, price: 20 }
];

🔸 Your Task to Perform

  • Multiply price × quantity for each item and add them up.
  • Return the total amount as a string, prefixed with "₹" (Indian Rupee).
  • If any price or quantity is invalid (not a number or ≤ 0), skip that item.

🧪 Example Output: 

"₹170" // Total Bill Amount of User

🔸 Bonus Tip for You

  • Use Array.prototype.reduce() to cleanly sum the totals.
  • Format the final output using template literals or Intl.NumberFormat if needed.

Day 18 JavaScript Challenge — “Strictly Same Elements”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re given two arrays. Write a function that checks whether both arrays contain the same elements at the same indexes — and with the same type.

Return true only if length, value, and types are all equal at each position.

🧪 Example Input:

const arr1 = [1, "2", 3];
const arr2 = [1, "2", 3];

🔸 Your Task to Perform

Compare both arrays:

  • Same length?
  • Each index: same value & type?

If all conditions pass, return true. Else return false.

🧪 Example Output:

true // same value & type at every position

🔸 Bonus Tip for You

Use typeof and === for strict checks.

Edge case: If either array is null or not actually an array — handle it gracefully.

Day 62 JavaScript Challenge — “Longest Substr Without Repeat”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a string, return the length of the **longest substring** that contains all unique characters. This problem is often used to test your ability to work with **sliding windows** and sets/maps in JavaScript.

🧪 Example Input:

const input = "abcabcbb";

🔸 Your Task to Perform

  • Use two pointers to define a sliding window of non-repeating characters.
  • Use a Set or Map to keep track of the characters in the current window.
  • Return the maximum length found as you slide the window.

🧪 Expected Output:

// Output: 3 ("abc")

🔸 Bonus Tip for You

  • Practice with edge cases: empty string, all same characters, string with spaces/symbols.
  • Try writing both brute-force and optimized versions to understand the time savings.

Day 46 JavaScript Challenge — “Movie Ticket Counter”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're working at a movie theatre. You receive a list of customers who bought tickets. Each customer has a name and the number of tickets they bought.

Your job is to calculate the total number of tickets sold and identify any customer who bought more than 5 tickets — they’re resellers and need to be flagged.

🧪 Example Input:

const customers = [
  { name: "Amit ", tickets: 2 },
  { name: "Sneha", tickets: 6 },
  { name: "Karan", tickets: 3 },
  { name: "Pooja", tickets: 8 },
];

🔸 Your Task to Perform

  • Loop through the customer list and calculate the total number of tickets.
  • Print the name of every customer who bought more than 5 tickets with a warning message.

🧪 Example Output:

Total Tickets Sold: 19
⚠️ Reseller Alert: Sneha
⚠️ Reseller Alert: Pooja

🔸 Bonus Tip for You

Don’t use reduce(). Try using simple for or for-of loops for better loop logic building.

Want a challenge? Add logic to handle if the same name appears twice — sum their ticket counts together before flagging.

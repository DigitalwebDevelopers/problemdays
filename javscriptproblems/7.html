Day 7 JavaScript Challenge — “Remove Duplicates”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given an array that may contain duplicate numbers. Your task is to remove all duplicates and return a new array that contains only unique elements — but in the **same order** as original.

🧪 Example Input: 


const input = [1, 3, 2, 3, 4, 1, 5];

🔸 Your Task to Perform

Loop through the array and track which numbers have already appeared.

Build a new array with only the first occurrence of each number.

Return that array.

🧪 Example Output: 

[1, 3, 2, 4, 5] // Fresh Array Updated

🔸 Bonus Tip for You

Don't use `new Set()` — solve it with pure loops and logic.

Maintain the original order — i.e., keep only the first occurrence of each element.

Day 30 JavaScript Challenge — “Deep Key Finder”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a deeply nested object, write a function to find and return the value of a specified key — no matter how deeply nested it is.

🧪 Example Input:

const data = {
  user: {
    info: {
      profile: {
        name: "Ankit",
        age: 24,
      },
    },
  },
};

findKeyDeep(data, "age");

🔸 Your Task to Perform

Traverse the object recursively. If the key matches, return its value. If it’s an object, search deeper. If nothing found, return "Not Found".

🧪 Example Output:


24 // This is value of the "age"

🔸 Bonus Tip for You

Use recursion to explore deeply nested levels.

Handle arrays inside the object too. Base case: not an object → skip. Loop through keys and go deep if value is object.

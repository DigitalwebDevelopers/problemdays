Day 11 JavaScript Challenge — “Group by Data Type”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a mixed array, return an object where keys are data types and values are arrays of all items of that type.

🧪 Example Input:

const input = [1, "hello", true, 42, "world", false, null, undefined, {}, [], () => {}];

🔸 Your Task to Perform

Loop through the array and check type of each item using typeof (and for arrays use Array.isArray()).

Group them in an object like: 

{
  number: [1, 42],  string: ["hello", "world"],
  boolean: [true, false], object: [null, {}],
  undefined: [undefined], array: [[]],
  function: [() => {}]
}

🧠 Bonus Tip for You

Try solving it without using .reduce() if you want to stay loop-pure.

Handle null and [] properly: typeof null === "object" and Array.isArray([]) is true.

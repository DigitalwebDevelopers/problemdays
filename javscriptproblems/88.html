Day 88 JavaScript Challenge — “Derangement Counter”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

A derangement is a permutation of elements in which no element appears in its original position.

You're given n people, each originally assigned to a numbered seat. Write a function that returns the number of ways the seats can be rearranged so that no person ends up in their original seat.

Use recursion for the logic. (Optional: Add memoization to boost performance!)

🧪 Sample Input:

countDerangements(3)

🧪 Expected Output:

2
// For 3 people: [2,3,1] and [3,1,2] are valid derangements.

🔸 Constraints

  • Input n will be a positive integer ≤ 10

🧠 Bonus Insight:

  • Base cases: D(1) = 0 , D(2) = 1
  • The recurrence relation: D(n) = (n - 1) * (D(n - 1) + D(n - 2))

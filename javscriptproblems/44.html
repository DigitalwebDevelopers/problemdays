Day 44 JavaScript Challenge — “Library System using Classes”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Create a simple library system using JavaScript classes.

You should be able to add books, issue books, and return books. A book should have a title, author, and availability status.

🧪 Example Features:

// Expected Methods:
library.addBook(new Book("Atomic Habits", "James Clear"));
library.issueBook("Atomic Habits");
library.returnBook("Atomic Habits");

🔸 Your Task to Perform

  • Create a Book class with title, author, and isAvailable properties.
  • Create a Library class with methods: addBook, issueBook, and returnBook.

🧪 Sample Output:

"Book issued successfully!"
"Book returned!"
"Book not found!"

🔸 Bonus Tip for You

Use constructor methods to initialize properties.

Try using Map or Array for book storage and find().

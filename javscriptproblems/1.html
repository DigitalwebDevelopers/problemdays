Day 1 JavaScript Challenge — "Decode the Weird Array" [ ]

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re given an array of mixed types — numbers, strings, nested arrays, and even functions! Your job is to:

1.   Traverse the array deeply. { Analise Code }

2.   Extract and print only the numbers.

3.   Ignore strings, functions, booleans, etc.

4.   Flatten any nested arrays while doing this (recursively).

🧪 Example Input: 

// this is the your Array for Traversing
const input = [ 1, "hello",
  [2, "world", [3, () => 5]],
  true,
  () => 100,
  [4, ["ignore me", 5]]
];

🧪 Expected Output: 

[1, 2, 3, 4, 5] // You have to find This

Functions that return a number when called should also be executed and included.

🔸 Bonus Points for You :

Sort final output in ascending order.

Write it in a pure functional style (no loops, only recursion and array methods like map, filter, flatMap).

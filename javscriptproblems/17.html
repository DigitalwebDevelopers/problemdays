Day 17 JavaScript Challenge — “Last Index Matcher”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re given a string and a target character. Find the last index where this character occurs. If the character does not exist in the string, return -1.

🧪 Example Input:

const str = "javascript is cool";
const target = "o";

🔸 Your Task to Perform

Loop through the string from end to start. As soon as you find the target character, return that index.

Do not use .lastIndexOf() or any built-in search method. Pure loop se karna hai.

🧪 Example Output:

17 // because 'o' last appears at index 17

🔸 Bonus Tip for You

Think in reverse: start your loop from the end of the string. First match = last index!

Try with special characters too like "!" or "#" — should still work.

Day 92 JavaScript Challenge — “Maximum Product Subarray”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're working on a financial app where users track profits over time. You're given an array of integers representing daily profit or loss.

Your task is to find the **maximum product** of a **contiguous subarray**. It may include both positive and negative numbers.

🧪 Sample Inputs:

maxProduct([2, 3, -2, 4]);         // Output: 6    → subarray: [2, 3]
maxProduct([-2, 0, -1]);           // Output: 0    → subarray: [0]
maxProduct([-2, 3, -4]);           // Output: 24   → subarray: [3, -4]

⚙️ Constraints:

  • Array length ≥ 1 and ≤ 10⁴
  • Elements range between -10 and 10 (inclusive)

🧠 Bonus Insight:

  • Unlike max sum subarray (Kadane’s Algorithm), here you must track both max and min products since negative × negative = positive.
  • Useful for analyzing streak-based metrics like profits, stock prices, or temperature swings.

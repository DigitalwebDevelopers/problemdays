Day 93 JavaScript Challenge — “Matrix Multiplication”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re building a scientific calculator app that supports linear algebra operations. One key feature is matrix multiplication.

Given two 2D arrays A and B, write a function to return the result of their matrix product.

🧪 Sample Inputs:


const A = [[1, 2],[3, 4]];

const B = [[2, 0],[1, 2]];

// Output:
[[4, 4],[10, 8]]

⚙️ Constraints:

  • The number of columns in A must match the number of rows in B
  • All values are integers from -100 to 100

🧠 Bonus Insight:

  • This problem is widely used in graphics engines, neural networks, and physics simulations.
  • Make sure you use three nested loops — two for the output matrix and one for the dot product calculation.

Day 49 JavaScript Challenge — “Movie ShowTime Validator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You run an online movie ticket system. Users can only book tickets for shows that are scheduled for today or in the future.

You are given a list of showtimes with dates in YYYY-MM-DD format. Your job is to filter out any shows that are already past (older than today).

🧪 Example Input:

const shows = [
  { title: "Avengers", date: "2024-12-30" },
  { title: "Inception", date: "2023-08-01" },
  { title: "Interstellar", date: "2025-06-25" },
];

🔸 Your Task to Perform

  • Use today's date (using new Date())
  • Loop through the shows
  • Return only those shows whose date is today or in the future

🧪 Sample Output:

[
  { title: "Avengers", date: "2024-12-30" },
  { title: "Interstellar", date: "2025-06-25" }
]

🔸 Bonus Tip for You

Remember to convert both dates to the same format (e.g. timestamp) before comparing.

Assume user's local timezone unless specified otherwise.

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given an array of mixed data types.

Your task is to extract all elements that occur only once AND are numbers.

🧪 Example Input: 

const input = [1, "2", 2, 1, 3, "3", 4, true, 4, 5];

🔸 Your Task to Perform

Loop through the array. Filter out all elements that are not numbers (e.g., strings, booleans).

From the remaining numbers, find those that occur only once. Return or log the result array.

🧪 Example Input: 

[2, 3, 5] // This may be your Output

🔸 Bonus Tip for You

Do not mutate the original array

Don’t use filter() or reduce() — solve using pure loops only

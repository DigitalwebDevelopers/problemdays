Day 22 JavaScript Challenge — “Find Missing Number”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given an array containing n distinct numbers from 1 to n+1, but exactly one number is missing. Find the missing number.

🧪 Example Input:

const arr = [1, 2, 4, 5];

🔸 Your Task to Perform

Find the missing number using loop. Don't sort or use extra arrays.

Use either math trick: sum formula or loop-based subtraction logic.

🧪 Example Output:

3 // Because 3 is missing

🔸 Bonus Tip for You

Use this sum formula: (n+1) * (n+2) / 2 to get expected total, and subtract array sum from it.

Day 45 JavaScript Challenge — “Parking Ticket Scanner”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given a list of vehicle entries and exits from a parking lot. Each vehicle logs its entry and exit times in hours (24-hr format). Your task is to print out which vehicles stayed more than 2 hours in the lot — they should be fined!

🧪 Example Input:

const logs = [
  { vehicle: "DL01AB1234", in: 9, out: 11 },
  { vehicle: "HR29CD9988", in: 10, out: 13 },
  { vehicle: "UP32XY4567", in: 14, out: 15 },
];

🔸 Your Task to Perform

  • Use a loop to go through each log.
  • If parking duration is more than 2 hours, print a message: "⚠️ Fine for: [Vehicle Number]"
  • Ignore vehicles with time less than or equal to 2 hours.

🧪 Example Output:

⚠️ Fine for: HR29CD9988

🔸 Bonus Tip for You

Don't use filter or map. Stick to basic loops (like for or for-of).

You can add conditions like "₹50 per extra hour" if you want to challenge yourself.

Day 29 JavaScript Challenge — “Anagram Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given two strings, check if one string is an anagram of the other. An anagram means both strings contain the same characters, just in a different order.

🧪 Example Input:

const str1 = "listen";
const str2 = "silent";

🔸 Your Task to Perform

Count how many times each letter appears in both strings. If both maps (or frequency counts) match — it's an anagram.

Strings can contain lowercase letters only. Do not use built-in sort method — do it manually.

🧪 Example Output:

true // because 'listen' and 'silent' have same letters

🔸 Bonus Tip for You

Try solving using object as character map — count up for str1, count down for str2.

Edge case: unequal lengths = auto false. Extra characters = false.

Day 27 JavaScript Challenge — “First Repeating Character”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a string, find the first character that repeats more than once. Return it. If no repetition found, return "No repetition".

🧪 Example Input:

const input = "javascriptlogics";

🔸 Your Task to Perform

Traverse the string from left to right. Store seen characters in a Map or Object.

Return the first character that occurs again.

🧪 Example Output:

a // because "a" is the first character that appears more than once

🔸 Bonus Tip for You

Use an object as a frequency map or set. This is called hashing in DSA.

Edge case: string like "abc" should return "No repetition"

Day 96 JavaScript Challenge — “First Unique Character Finder”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re building a real-time messaging app. In a given chat message (a string), you want to highlight the first non-repeating character to help detect unusual or new message patterns.

Your task is to find the index of the first unique character. If no such character exists, return -1.

🧪 Sample Input:


const message = "levelup";

// Output:
0   // 'l' is the first unique character

💡 Explanation:

  • 'l' appears only once and comes before all other unique characters.
  • If all characters repeat (e.g., "aabbcc"), then return -1.

🔸 Real-World Use Case:

  • Unique character detection is crucial in sentiment analysis, spell-check systems, or building secure identifiers.

Day 55 JavaScript Challenge — “Capitalize Each Word in Sentence”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given a sentence as a string. Your task is to capitalize the first letter of each word, while keeping the rest of the characters in lowercase.

🧪 Example Input: 

const input = "hello world from javascript";

🔸 Your Task to Perform

  • Split the sentence into words using space.
  • Capitalize the first character of each word and make the rest lowercase.
  • Join the words back into a sentence and return the result.

🧪 Example Output: 

"Hello World From Javascript"

🔸 Bonus Tip for You

  • Ensure your function ignores extra spaces and handles mixed-case inputs.
  • Avoid using toTitleCase() as JavaScript doesn’t natively have it — practice using `split`, `map`, and string manipulation.

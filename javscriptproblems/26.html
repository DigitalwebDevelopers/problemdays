Day 26 JavaScript Challenge — “Frequent After K Removal”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of integers and a number k, remove exactly k elements from the array in such a way that the element with the highest frequency remains (or becomes) most frequent in the remaining array. Return that element.

🧪 Example Input: 

const nums = [4, 3, 1, 1, 2, 3, 3, 4];
const k = 3;

🔸 Your Task to Perform

Count frequency of each number. Remove k elements in such a way that **any number becomes most frequent**.

Return that number which will have highest frequency after these k removals.

🧪 Example Output: 

3 // removing two 4s and one 1, 3 becomes most frequent

🔸 Bonus Tip for You

Use Map to count frequencies and think greedy: remove least frequent elements first.

This problem is also a base version of a heap-based greedy optimization.

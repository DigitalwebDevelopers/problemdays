Day 4 JavaScript Challenge — “Deep Count”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given a nested array (array inside array). Your task is to count how many total elements (non-array) are present,

no matter how deep the nesting is.

🧪 Example Input: 

const input = [1, [2, [3, 4], 5], 6];

🔸 Your Task to Perform

Loop through the array and go inside nested arrays too using recursion.

Count how many non-array elements (like numbers, strings, booleans) are there in total.

Return that count.

🧪 Example Output: 

6 // Total elements = 1, 2, 3, 4, 5, 6

🔸 Bonus Tip for You

Use recursion to handle deeply nested arrays.

Don’t use .flat() or .toString()

Count all non-array elements including strings and booleans.

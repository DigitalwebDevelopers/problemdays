Day 84 JavaScript Challenge — “Jacobsthal Number Generator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're contributing to a pattern recognition system that models growth and decay rates using alternative number sequences.

Your task is to implement a function that returns the nth Jacobsthal number.

The sequence is defined recursively as:

  • J(0) = 0
  • J(1) = 1
  • J(n) = J(n − 1) + 2 × J(n − 2)

This sequence grows differently from Fibonacci, and it's useful in circuits and logic design.

🧪 Sample Input:

jacobsthal(5) ➞ 11
// Sequence: 0, 1, 1, 3, 5, 11, ...

🧪 Output:

11

💡 Bonus Tip:

  • Use dynamic programming or iteration for performance if n > 25.
  • This sequence appears in digital signal processing and recursive state models.

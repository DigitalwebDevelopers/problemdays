Day 16 JavaScript Challenge — “Find Most Frequent Character”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re given a string. Find the character that appears the most number of times. Ignore spaces and consider lowercase and uppercase as different characters.

🧪 Example Input:

const str = "Knock Knock JavaScript";

🔸 Your Task to Perform

Traverse the string and count frequency of each character (excluding spaces).

Then return the character with the highest frequency.

If two characters tie, return the first one that reaches max count.

🧪 Example Output:

"k" // appears 3 times (case-sensitive)

🔸 Bonus Tip for You

Use an object to store character frequencies, then loop through it to find the most frequent one.

Edge case: if string is empty or contains only spaces — return "No valid character".

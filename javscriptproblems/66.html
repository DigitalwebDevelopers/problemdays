Day 66 JavaScript Challenge — “Filter Out Duplicate Emails”

Taught By Code with Deepanshu Knox

📧 Problem Of the Day —

Imagine you're building a newsletter signup system. A list of email addresses may contain duplicates (case-insensitive). Your job is to return a new array with **only unique email addresses**, preserving their original order.

🧪 Example Input:

const emails = [
  "user1@example.com", "User1@example.com",
  "admin@site.com", "test@demo.com",
  "ADMIN@site.com"
];

🔸 Your Task to Perform

  • Remove all duplicates regardless of case.
  • Output the filtered array of emails, maintaining their original order from input.

🧪 Expected Output:

[
  "user1@example.com",
  "admin@site.com", "test@demo.com"
]

🔸 Bonus Tip for You

  • Use a Set to track lowercased emails you've already seen.
  • Preserve order by looping and checking before pushing to the result.

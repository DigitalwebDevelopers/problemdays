Day 36 JavaScript Challenge — “Numbers in Nested Object”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a deeply nested object, extract all numeric values and return them inside an array.

🧪 Example Input: 

const input = {
  a: 5,
  b: {
    c: "hello",
    d: {
      e: 12,
      f: {
        g: 90
      }
    },
    h: false
  },
  i: "skip me"
};

🔸 Your Task to Perform

  • Use recursion to explore all keys.
  • Push only numbers into an array.
  • Return the final array at the end.

🧪 Expected Output: 

[5, 12, 90]

🔸 Bonus Tip for You

You can also add a feature to return numbers in sorted order for extra challenge.

Avoid using JSON.stringify() for solving this. This challenge is about **recursive traversal**, not string search.

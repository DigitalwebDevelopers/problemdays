Day 75 JavaScript Challenge — “Event Conflict Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Imagine you're building a calendar scheduling app. A user inputs multiple events with a start and end time.

Your task is to check whether any of the events overlap with each other. Return true if there's a conflict, or false if all events are non-overlapping.

🧪 Sample Input:

const events = [
  { title: "Team Meeting", start: 9, end: 10 },
  { title: "Client Call", start: 10, end: 11 },
  { title: "Code Review", start: 10.5, end: 11.5 }
];

🔸 Expected Output:

// Overlap between Client Call and Code Review
true

🔸 Bonus Tip

  • Sort the events by start time first — then check if any event’s start is less than the previous event’s end.
  • This logic is used in calendar apps like Google Calendar or Outlook.

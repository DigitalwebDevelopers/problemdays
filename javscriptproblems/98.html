Day 98 JavaScript Challenge — “Jewels and Stones”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Imagine you’re building a gem identification system where some stones are considered "jewels" and others are just regular stones.

You're given two strings: jewels representing types of jewels, and stones representing what you have.

Each character in both strings is a type of stone. Your task is to determine how many of the stones are jewels.

🧪 Sample Input:


const jewels = "aA";
const stones = "aAAbbbb";

// Output:
3

💡 Explanation:

  • There are 2 'A's and 1 'a' in stones which are jewels. So total = 3.

🔸 Real-World Use Case:

  • Such logic can help in filtering premium items in e-commerce apps, or prioritizing certain tags (like VIP users, priority products) during searches.

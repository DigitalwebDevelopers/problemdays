Day 97 JavaScript Challenge — “Count the Good Pairs”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Imagine you're building an online analytics dashboard where you're analyzing repeating behaviors.

You’re given an array of integers. Your task is to count the number of “good pairs” – pairs where:

  • nums[i] === nums[j]
  • and i < j

🧪 Sample Input:


const nums = [1, 2, 3, 1, 1, 3];

// Output:
4

💡 Explanation:

  • The good pairs are: (0,3), (0,4), (3,4), and (2,5)

🔸 Real-World Use Case:

  • Good pair detection can be useful in event tracking systems, frequency analysis in user activity, or identifying repeating patterns in time series data.

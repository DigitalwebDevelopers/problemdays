Day 32 JavaScript Challenge — “Object Number Extractor”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array that can contain numbers, objects, arrays, booleans, and functions — extract only number values

from anywhere, including values inside objects and arrays. Ignore all other data types.

🧪 Example Input: 

const input = [
  1,
  { a: 2, b: "hello" },
  [3, { c: 4 }],
  true,
  () => 5,
  { d: [6, "world"] },
];

🔸 Your Task to Perform

  • Traverse all items in the array, including nested arrays and object values.
  • Return only numbers found anywhere inside.
  • Ignore functions, booleans, and strings.

🧪 Expected Output: 

[1, 2, 3, 4, 6]

🔸 Bonus Tip for You

You'll need to check for objects and arrays recursively. Use typeof and Array.isArray() smartly.

Avoid using JSON.stringify or similar shortcuts — solve it like a mini parser!

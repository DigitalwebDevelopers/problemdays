Day 8 JavaScript Challenge — “Ends With Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Write a function that checks whether a given string ends with a given target string. But you are NOT allowed to use .endsWith() or .slice() or .substring().

🧪 Example Input: 

const str = "KnockTheCode";
const target = "Code";

🔸 Your Task to Perform

Loop through the original string and manually check if the last characters match the target.

Return true if it ends with the target string, else return false.

No use of built-in methods like .endsWith(), .slice(), etc.

🧪 Example Output: 

true // Because str ends with Target

🔸 Bonus Tip for You

Use manual character comparison via for loop and str[i].

Handle edge cases: empty strings, or target longer than main string.

Challenge yourself to write the logic in one loop only.

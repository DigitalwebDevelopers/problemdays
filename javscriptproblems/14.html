Day 14 JavaScript Challenge — “Isomorphic Strings”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given two strings s1 and s2, determine if they are isomorphic. Two strings are isomorphic if characters in s1 can be replaced to get s2, with one-to-one mapping.

🧪 Example Input:

const s1 = "egg";
const s2 = "add";

🔸 Your Task to Perform

Loop through both strings simultaneously.

Use two maps to track the character mapping from s1 → s2 and s2 → s1.

Return true if mapping is consistent for all characters. Else return false.

🧪 Example Output:

true // "e" maps to "a", "g" maps to "d"

🔸 Bonus Tip for You

Don’t use Set or filter — stick with raw object or Map() logic.

Try with edge cases like: unequal lengths, same letters mapping to different ones etc.

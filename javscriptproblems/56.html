Day 56 JavaScript Challenge — “Sort Employees by Salary”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are managing employee records. Each employee has a name and a salary. Your task is to sort the list of employees in descending order based on their salaries.

🧪 Example Input: 

const employees = [
  { name: "Aditi", salary: 50000 },
  { name: "Vikram", salary: 75000 },
  { name: "Riya", salary: 60000 }
];

🔸 Your Task to Perform

  • Sort the objects in the array by their salary in descending order.
  • Avoid using any external libraries — use `.sort()` method in JavaScript.

🧪 Example Output: 

[
  { name: "Vikram", salary: 75000 },
  { name: "Riya", salary: 60000 },
  { name: "Aditi", salary: 50000 }
]

🔸 Bonus Tip for You

  • What if the salary is a string like "75,000"? Convert it to a number first using `.replace()` and `parseInt()` before sorting.
  • Handle cases where two employees have the same salary.

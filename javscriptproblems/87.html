Day 87 JavaScript Challenge — “Fibonacci Number Generator 🌀”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

The Fibonacci sequence is defined as: F(n) = F(n - 1) + F(n - 2), with base cases: F(0) = 0, F(1) = 1.

Your task is to write two versions of a function to return the nth Fibonacci number:
1. One using recursion
2. One using iteration

🧪 Sample Input:

fibonacciRecursive(8)
fibonacciIterative(8)

🧪 Expected Output:


21 // Both methods should return this for n = 8

📌 Notes:

  • Recursive method should clearly show base case and recursive step.
  • Iterative method should use a simple loop and two variables to track last two values.

🧠 Challenge Tip:

  • Try comparing the performance for fibonacciRecursive(40) vs iterative version.
  • Consider adding memoization to improve recursive performance.

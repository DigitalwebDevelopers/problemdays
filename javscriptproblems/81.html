Day 81 JavaScript Challenge — “Tribonacci Numbers Generator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

In the Tribonacci sequence, each term is the sum of the previous three terms. It starts with three predetermined values like [0, 1, 1].

You're designing a number generator tool for a math learning platform. The platform needs the nth number in the Tribonacci sequence.

Your job is to implement a function that:

  • Takes a number n as input
  • Returns the nth Tribonacci number based on the sequence [0, 1, 1]

🧪 Sample Input:

tribonacci(6) ➞ 11
// Sequence: 0, 1, 1, 2, 4, 7, 11...

🧪 Output:

11

💡 Hints & Concepts:

  • Use iteration (looping) instead of recursion for better performance.
  • Maintain a rolling window of the last 3 numbers.

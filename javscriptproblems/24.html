Day 24 JavaScript Challenge — “Flatten the Deep Array”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a deeply nested array, flatten it into a single-level array using recursion (no built-in flat() allowed).

🧪 Example Input:

const arr = [1, [2, [3, [4, [5]]]], 6];

🔸 Your Task to Perform

Traverse each item. If it's a number, push to final result. If it's an array, call your function recursively on it.

Do not use `.flat()` or external libraries — only recursion or loops.

🧪 Example Output:

[1, 2, 3, 4, 5, 6]

🔸 Bonus Tip for You

Test it on very deeply nested arrays. This builds your confidence in recursion and edge case safety.

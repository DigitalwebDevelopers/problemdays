Day 70 JavaScript Challenge — “Compute Averages for Readings”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're building a weather analysis tool that receives hourly temperature readings. For each reading, compute the running average of the last 3 hours (including current), returning a new array where each element represents that 3-hour average.

🧪 Example Input:

const temps = [20, 22, 21, 19, 24, 20];

🔸 Your Task to Perform

  • Loop through `temps` and at each index compute average of current and previous two values.
  • For the first two entries, compute the average of available readings only.
  • Return a new array of floating-point averages, rounded to 2 decimal places.

🧪 Expected Output:

[20.00, 21.00, 21.00, 20.67, 21.33, 21.00]

🔸 Bonus Tip for You

  • Use a sliding window with running sum instead of recomputing sum in each iteration.
  • This pattern is used in analytics dashboards, streaming data processing, and IoT sensor data smoothing.

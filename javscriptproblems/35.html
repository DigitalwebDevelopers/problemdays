Day 35 JavaScript Challenge — “Object Key Filter”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given an object with multiple key-value pairs. Return a new object that only includes those key-value pairs where the value is a number.

🧪 Example Input: 

const input = {
  name: "Alice",
  age: 25,
  verified: true,
  score: 87,
  city: "Delhi"
};

🔸 Your Task to Perform

  • Loop through all keys in the object.
  • Check if the value is a number using typeof.
  • Push valid key-value pairs into a new object.

🧪 Expected Output: 

{ age: 25, score: 87 }

🔸 Bonus Tip for You

Don't use Object.entries() or Object.keys() for this task — loop manually using for...in for better grasp.

Optional: Extend it to include floating-point checks and remove `NaN`.

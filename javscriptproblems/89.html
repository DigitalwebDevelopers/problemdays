Day 89 JavaScript Challenge — “Catalan Number Calculator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Catalan numbers form a sequence that has wide applications in mathematics, such as counting valid parentheses, binary search trees, and more.

Given a number n, write a function to return the n-th Catalan number.

🧪 Sample Input:

getCatalan(4)

🧪 Expected Output:

14 // Catalian pattern when N = 4

🔸 Formula to Use:

  • C(n) = (2n)! / ((n + 1)! * n!)
  • Use recursion or iterative factorial computation.

🧠 Bonus Insight:

  • C(0) = 1, C(1) = 1, C(2) = 2, C(3) = 5, C(4) = 14, C(5) = 42 ...
  • Used in counting valid combinations of nested brackets and paths in binary trees.

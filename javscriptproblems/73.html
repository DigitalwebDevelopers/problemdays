Day 73 JavaScript Challenge — “Flight Seat Allocation Check”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are working on a flight booking system. When users choose their preferred seat numbers, your system must ensure no duplicate seat is selected and all seat numbers are valid (between 1 and 60).

Write a function that takes an array of seat numbers and returns either “All seats are valid” or an error object describing what went wrong.

🧪 Sample Input:

const userSeats = [12, 25, 40, 60];

🔸 Rules to Implement:

  • All seat numbers must be unique (no duplicate values).
  • All seats must be between 1 and 60 inclusive.

🧪 Output:

// Valid case
"All seats are valid"

// Invalid case
{
  duplicates: [25],
  invalid: [77, -2]
}

🔸 Bonus Tip

  • Use Set to check for duplicates, and simple range conditions to validate seat limits.
  • This is a common scenario in online ticketing apps — practice like it's real!

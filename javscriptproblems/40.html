Day 40 JavaScript Challenge — “Find Deepest Key”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given a deeply nested JavaScript object. Your task is to return the key which lies at the deepest level of nesting.

🧪 Example Input: 

const data = {
  a: {
    b: {
      c: {
        d: {
          target: 42,
        },
      },
    },
  },
};

🔸 Your Task to Perform

  • Write a recursive function
  • Keep track of depth and key name
  • Return the key with the deepest nesting

🧪 Expected Output: 

"target"

🔸 Bonus Tip for You

You may find it useful to pass current depth and deepest key found so far as arguments in your recursive calls.

Day 64 JavaScript Challenge — “Search by Name Prefix”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are building a user search feature for a messaging app. Given a list of user objects and a search prefix (a string), return an array of full names that **start with the given prefix** (case-insensitive).

🧪 Example Input:

const users = [
  { name: "Amit Sharma", id: 1 },
  { name: "Ankit Rana", id: 2 },
  { name: "Ravi Kumar", id: 3 },
  { name: "amisha Sethi", id: 4 }
];

const searchPrefix = "am";

🔸 Your Task to Perform

  • Filter the array and return names that start with the given prefix (ignore case).
  • If no name matches, return an empty array.

🧪 Expected Output:

["Amit Sharma", "amisha Sethi"]

🔸 Bonus Tip for You

  • Use `.toLowerCase()` on both the prefix and name to ensure case-insensitive matching.
  • Use `.startsWith()` method instead of regex for better readability.

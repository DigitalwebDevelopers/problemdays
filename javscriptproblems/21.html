Day 21 JavaScript Challenge — “Clean and Count Words”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re given a string that may contain extra spaces, special characters, and punctuation. Your task is to clean the string (remove symbols & extra spaces), split it into words, and return how many valid words exist.

🧪 Example Input:

const text = "  Hello! This... is     JavaScript, isn't it?  ";

🔸 Your Task to Perform

Remove punctuation, trim spaces, normalize all words to lowercase (optional), and then count the number of words.

🧪 Example Output:

7 // Words are: "Hello", "This", "is", "JavaScript", "isn't", "it"

🔸 Bonus Tip for You

Use `.replace()` with RegEx to clean the string. Then `.split(" ")` and `.filter()` to count non-empty parts.

Day 68 JavaScript Challenge — “Merge Available Time Slots”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re building a meeting scheduler. Users input their available time slots as an array of intervals (each with a start and end time in 24-hour minutes). Your task is to merge overlapping or adjacent time slots to show the final combined availability.

🧪 Example Input:

const slots = [
  { start: 540, end: 600 },  // 9:00–10:00
  { start: 580, end: 630 },  // 9:40–10:30
  { start: 700, end: 760 }   // 11:40–12:40
];

🔸 Your Task to Perform

  • Sort the intervals by start time.
  • Merge overlapping or touching intervals (e.g., one ends at 600, next starts at 600).
  • Return a new array of cleaned, merged intervals.

🧪 Expected Output:

[
  { start: 540, end: 630 },
  { start: 700, end: 760 }
]

🔸 Bonus Tip for You

  • Loop through sorted slots and merge using an output array for efficiency.
  • This helps in calendars, booking apps, or resource scheduling systems.

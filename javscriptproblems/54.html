Day 54 JavaScript Challenge — “Check if Sentence is Pangram”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Write a function that checks whether a given sentence is a pangram. A pangram is a sentence that contains every letter of the English alphabet at least once.

🧪 Example Input: 

const input = "The quick brown fox jumps over the lazy dog";

🔸 Your Task to Perform

  • Ignore case sensitivity and whitespace.
  • Return true if the sentence is a pangram, otherwise false.

🧪 Example Output: 

true

🔸 Bonus Tip for You

  • Use a Set to track all unique alphabet characters.
  • Avoid regex if you're solving for interviews – use clean loops.

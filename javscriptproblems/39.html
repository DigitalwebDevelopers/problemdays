Day 39 JavaScript Challenge — “Find Oldest Person From List”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are given an array of people with their names and birthdates (in string format). Your task is to return the name of the oldest person.

🧪 Example Input: 

const people = [
  { name: "Amit", dob: "1995-03-15" },
  { name: "Kavya", dob: "1992-07-01" },
  { name: "Rahul", dob: "1998-11-20" },
];

🔸 Your Task to Perform

  • Loop through the list of people
  • Convert string date to JavaScript Date object
  • Find the person with the earliest (oldest) date

🧪 Expected Output: 

"Kavya" // Born in 1992

🔸 Bonus Tip for You

You can compare two date objects using simple `<` or `>` operators. And if you want extra learning, try building a comparator for sorting by date.

Day 9 JavaScript Challenge — “Next Day Calculator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a date in the form of day, month, and year, write a function to calculate the next calendar day.

Do not use the built-in Date object. All logic must be done manually.

🧪 Example Input: 

const month = 2;
const day = 28;
const year = 2024;

🔸 Your Task to Perform

Handle month-end transitions like 31 → 1 (next month), or 28/29 for February (leap year logic).

Correctly roll over the month and year as needed.

Return the next date as an object: { day, month, year }

🧪 Example Output: 

{ day: 29, month: 2, year: 2024 }

🔸 Bonus Tip for You

Carefully handle leap year logic (year divisible by 4, but century rules apply too).

No new Date() or toLocaleString — pura logic tu likhega.

Try to build a helper function like getDaysInMonth(month, year) for clean code.

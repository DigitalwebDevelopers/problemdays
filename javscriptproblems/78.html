Day 78 JavaScript Challenge — “Transaction Log Balancer”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are working on a digital wallet app. Each user has a transaction log that includes credits and debits as objects with a type ("credit" or "debit") and an amount (number).

Your task is to:

  • Calculate the final balance
  • Return an object showing total credits, total debits, and the final balance
  • If any entry has invalid type or non-numeric amount, ignore it

🧪 Sample Input:


const transactions = [
  { type: "credit", amount: 1000 }, { type: "debit", amount: 300 },
  { type: "credit", amount: 500 }, { type: "debit", amount: 200 },
  { type: "debit", amount: "oops" }, // ignore
  { type: "invalid", amount: 100 },  // ignore
];

🧪 Output:


{
  totalCredit: 1500,
  totalDebit: 500,  balance: 1000
}

🔸 Bonus Tip

  • Validate amount with typeof x === "number" && !isNaN(x)
  • Use reduce() to iterate over transaction logs

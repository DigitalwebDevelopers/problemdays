Day 76 JavaScript Challenge — “Online Store Stock Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re helping manage an online store’s inventory. When a customer tries to place an order, you need to check if the quantity requested is available.

Your task is to create a function that checks an order object against a stock object, and returns:

  • true if all items are in stock with enough quantity
  • Or an object showing which items are out of stock or short

🧪 Sample Input:

const stock = {
  "laptop": 5, "mouse": 10, "keyboard": 2
};

const order = {
  "laptop": 2, "mouse": 11, "keyboard": 2
};

🔸 Expected Output:

{
  mouse: "Only 10 in stock"
}

🔸 Bonus Tip

  • Useful in real-time stock validation on checkout pages.
  • Combine Object.keys, basic conditionals and loops to solve this.

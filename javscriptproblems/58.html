Day 58 JavaScript Challenge — “Validate HTML Tag Structure”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are building a lightweight HTML validator. The goal is to ensure that every opening tag has a matching closing tag, and the tags are properly nested.

🧪 Example Input: 

const htmlSnippet = "<div><p>Hello</p></div>";

🔸 Your Task to Perform

  • Parse the string and check whether each opening tag has a matching closing tag.
  • Tags must close in the correct nested order. Example: <div><p></p></div> is valid.
  • Tags like <br> or <img /> (self-closing) should be ignored or handled separately.

🧪 Example Output: 

true // Because the tags are balanced and properly nested

🔸 Bonus Tip for You

  • Use a stack to track opening tags. Pop when a correct closing tag is found.
  • Regular expressions can help extract tags efficiently: /<\/?\w+.*?>/g

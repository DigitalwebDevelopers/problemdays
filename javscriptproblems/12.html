Day 12 JavaScript Challenge — “Array Jump Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of integers where each element represents the max jump length at that position, return whether it's possible to reach the last index starting from the first index.

🧪 Example Input:


const input = [2, 3, 1, 1, 4];

🔸 Your Task to Perform

Starting from index 0, use the number at each index to determine how far you can jump.

Return true if you can reach the last index, otherwise return false.

The number represents max jump size from that position.

🧪 Example Output:


true // Because you can reach the end via jumps: 2 → 3 → 4

🧠 Bonus Tip for You

Try solving it in one loop. Track the maximum index you can reach.

Handle edge cases like [0, 2, 3] — should return false.

Day 83 JavaScript Challenge — “Double Factorial Calculator”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're working on a math toolkit for data analysts. One special function in this toolkit is the double factorial.

Your task is to write a function that calculates the double factorial of a given number n.

Double factorial is defined as:

  • If n is even: n × (n − 2) × (n − 4) × ... × 2
  • If n is odd: n × (n − 2) × (n − 4) × ... × 1

For example: 7!! = 7 × 5 × 3 × 1 = 105

🧪 Sample Input:

doubleFactorial(8) ➞ 384
// 8 × 6 × 4 × 2 = 384

🧪 Output:

384 // You have to find Answer

💡 Tips & Application:

  • Use a simple loop to decrement by 2 and multiply.
  • This operation is often used in combinatorics and probability.

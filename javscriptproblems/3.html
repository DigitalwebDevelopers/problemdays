Day 3 JavaScript Challenge — “First Unique Finder”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of integers, find the first number that occurs only once in the array.

🧪 Example Input: 

const input = [4, 5, 1, 2, 0, 4, 5, 2];

🔸 Your Task to Perform

Loop through the array. Track frequency of each number. Return the first number that occurs only once.

If no unique element exists, return "No unique number".

🧪 Example Input: 

1 // Because 1 is the Unique Number

🔸 Bonus Tip for You

Don’t use .filter() or .find() — use pure loops only

Try solving it in a single pass (advanced). Handle empty array edge case

Day 50 JavaScript Challenge — “Smart Cart Quantity Updater”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are building a shopping cart system. Customers add the same product multiple times. Each time a product is added, you need to update its quantity instead of duplicating it.

Given an array of cart items (each with id and quantity), merge the repeated products by ID and add their quantities together.

🧪 Example Input:

const cart = [
  { id: "p1", quantity: 2 },
  { id: "p2", quantity: 1 },
  { id: "p1", quantity: 3 },
  { id: "p3", quantity: 5 },
  { id: "p2", quantity: 2 }
];

🔸 Your Task to Perform

  • Loop through the array of cart items
  • Track quantities using a temporary object or map
  • Return a new array with unique items and updated quantities

🧪 Sample Output:

[
  { id: "p1", quantity: 5 },
  { id: "p2", quantity: 3 },
  { id: "p3", quantity: 5 }
]

🔸 Bonus Tip for You

Use object or Map to store product ID as key and quantity as value.

Keep your return array clean and sorted (optional).

Day 69 JavaScript Challenge — “Find Common Free Time”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re building a travel group coordination tool. Each traveler shares their busy schedule for the day as time intervals (in minutes). You need to compute the **common free time slots** where all members are available simultaneously.

🧪 Example Input:

const schedules = [
  [ { start: 540, end: 600 }, { start: 660, end: 720 } ], // Traveler A
  [ { start: 570, end: 630 } ], // Traveler B
  [ { start: 600, end: 660 } ]  // Traveler C
];
// Day window 9:00‑13:00 = 540–780 mins

🔸 Your Task to Perform

  • Determine each traveler's busy periods and compute shared free periods between 540–780.
  • Find all intervals where **no one is busy**—i.e., all have free time.

🧪 Expected Output:

[ { start: 630, end: 660 } ]
// All three are free between 10:30‑11:00

🔸 Bonus Tip for You

  • You can invert busy intervals into free slots per traveler, then compute their intersection.
  • Useful in calendar apps, meeting schedulers, and travel coordination tools.

Day 33 JavaScript Challenge — “Common Element Checker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given two arrays, check if there is at least one common element between them. Return true if found, otherwise false.

🧪 Example Input: 

const arr1 = [1, 2, 3, 4];
const arr2 = [9, 8, 4, 6];

🔸 Your Task to Perform

  • Loop through one array and check if any element exists in the other.
  • Don’t use .includes() or .some()
  • Use only basic loops or objects.

🧪 Expected Output: 

true // Because 4 is common in both arrays

🔸 Bonus Tip for You

Try to solve it using only one loop by storing elements from one array in an object (hash table).

Works faster on large arrays — welcome to O(1) lookup logic!

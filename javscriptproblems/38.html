Day 38 JavaScript Challenge — “Shuffle Check Two Strings”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given two strings, check if one string is a shuffle (or jumbled form) of the other string. Each character must be used exactly once.

🧪 Example Input: 

const str1 = "silent";
const str2 = "listen";

🔸 Your Task to Perform

  • Check if both strings have the same length.
  • Ensure both strings contain the same letters with the same frequency.

🧪 Expected Output: 

true // because "silent" is a shuffle of "listen"

🔸 Bonus Tip for You

Avoid using sort() if you want to practice using character maps or frequency objects manually. Also, handle case-sensitivity carefully.

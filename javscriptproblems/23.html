Day 23 JavaScript Challenge — “Find Most Frequent Word”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a paragraph of text, find the word that appears most frequently (ignore case and punctuation). Return both the word and its count.

🧪 Example Input:

const text = "Knock knock! Who's there? JavaScript. JavaScript who? JavaScript forever!";

🔸 Your Task to Perform

Clean the text (remove punctuation), convert to lowercase, split into words and use an object to count frequency.

Return the word with the highest frequency and how many times it occurred.

🧪 Example Output:

{ word: "javascript", count: 3 }

🔸 Bonus Tip for You

Use `.replace(/[^\w\s]/g, '')` to remove punctuation and `.toLowerCase()` to normalize case.

Day 52 JavaScript Challenge — “Smiley Faces Counter”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of strings, count how many valid smiley faces are present. A valid smiley face must follow these rules:

  • Eyes must be ":" or ";"
  • An optional nose can be "-" or "~"
  • Mouth must be ")" or "D"

🧪 Example Input: 

const smileys = [":)", ";(", ";}", ":-D", ":~)", ";~D", ":--)", ":D"];

🔸 Your Task to Perform

Loop through the array and count how many strings strictly match the smiley format. Ignore all other strings. Try using regular expressions or simple character checks — whichever feels right to you!

🧪 Expected Output: 

3 // Emojies Are Smiling

🔸 Bonus Tip for You

Try solving it using both regex and pure loop logic for practice. Avoid using external libraries.

Don’t confuse valid emojis with smiley faces. Focus on correct structure only.

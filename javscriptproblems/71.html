Day 71 JavaScript Challenge — “Find the K‑th Highest Score”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re analyzing response time scores from a server monitoring system. You need to find the K-th highest response time to evaluate performance thresholds.

🧪 Example Input:

const times = [120, 200, 150, 180, 130];
const k = 2;

🔸 Your Task to Perform

  • Return the 2nd highest response time (i.e., second-largest number).
  • Handle duplicates correctly (e.g., [5, 1, 5], k=1 → 5, k=2 → 5, k=3 → 1).
  • Edge cases: if k > length, return null or an error message.

🧪 Expected Output:

180 // K-th Highest Score

🔸 Bonus Tip for You

  • Use sorting with a Set to remove duplicates, then index: sortedUnique[k‑1].
  • Extra: Optimize using a MinHeap of size k (advanced).

Day 82 JavaScript Challenge — “Nth Fibonacci Finder”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're building a dashboard for a math education app that visualizes number sequences.

One of the features includes calculating the nth Fibonacci number. Your job is to implement a function that returns the nth value from the Fibonacci sequence where:

F(0) = 0, F(1) = 1, and F(n) = F(n-1) + F(n-2)

  • The input will be a non-negative integer n.
  • Output must be the nth number in the sequence.

🧪 Sample Input:

fibonacci(7) ➞ 13
// Sequence: 0, 1, 1, 2, 3, 5, 8, 13...

🧪 Output:

13

💡 Hints & Concepts:

  • Avoid recursion unless memoized — iterative approach is faster and safer.
  • This logic is helpful for time/space optimization in interviews.

Day 31 JavaScript Challenge — “Flatten and Filter Numbers”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a deeply nested array, write a function to extract all numbers while ignoring all other data types including strings, booleans, objects, and functions.

🧪 Example Input: 

const input = [
  1,
  "skip",
  [2, () => 3, [4, "nope", [5, true]]],
  false,
  [6, ["deep", [7]]]
];

🔸 Your Task to Perform

  • Traverse array recursively
  • Collect all items that are strictly numbers
  • Return flat array of numbers only

🧪 Expected Output: 

[1, 2, 4, 5, 6, 7]

🔸 Bonus Tip for You

Try solving using recursion and loops only — avoid using built-in .flat() or .flatMap() to get the most logic-building practice!

Think of how to handle an array that contains only non-number elements (edge case).

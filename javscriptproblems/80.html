Day 80 JavaScript Challenge — “The Notification Glitch”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're building a notification system where new messages should pop up every 2 seconds using setInterval. However, there's a bug — all notifications show the same message number: "Message #5" even though they were supposed to show Message #1, Message #2, and so on.

Your task:

  • Figure out why the wrong number is shown in each message
  • Fix the code so it logs the correct message number in each interval

🧪 Buggy Input Code:


function notify() {
  for (var i = 1; i <= 5; i++) {
    setTimeout(function () {
      console.log("📩 Message #" + i);
    }, i * 2000);
  }
}

notify();

🧪 Expected Output:


📩 Message #1
📩 Message #2
📩 Message #3
📩 Message #4
📩 Message #5

💡 Hints & Concepts:

  • Understand closures and how var vs let affects scoping
  • Use IIFE (Immediately Invoked Function Expression) or let to fix the bug

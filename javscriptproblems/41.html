Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of strings (names), build a function that returns only the items that match the user’s input.

Filtering should ignore case and work even if the user enters a partial match. ( anywhere in the string )

🧪 Example Input: 

const names = ["Sahil", "Ravi", "Ananya", "Ravindra", "Sakshi"];
const input = "ra"; // or "Ra"

🔸 Your Task to Perform

  • Ignore case sensitivity
  • Return all matching names that include the input
  • Use basic loops or .filter

🧪 Expected Output: 

["Ravi", "Ravindra"]

🔸 Bonus Tip for You

Try wrapping this logic inside a function like `filterNames(input)` so it can be reused easily inside live UI filtering or event listeners.

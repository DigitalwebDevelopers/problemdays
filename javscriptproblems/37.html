Day 37 JavaScript Challenge — “Build Grouped Words by Length”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of words, group them into an object where the key is the word's length and the value is an array of words having that length.

🧪 Example Input: 

const words = ["hi", "hello", "sun", "no", "world", "a"];

🔸 Your Task to Perform

  • Loop through the array of strings.
  • Group words by their length inside an object.
  • Don’t use reduce() or groupBy() (if available).

🧪 Expected Output: 

{
  1: ["a"],
  2: ["hi", "no"],
  3: ["sun"],
  5: ["hello", "world"]
}

🔸 Bonus Tip for You

This is a great chance to get comfortable with working with keys, conditions, and loops inside objects. Try optimizing how you check for existence of a key.

Day 95 JavaScript Challenge — “Flip 0 to Maximize Consecutives”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Imagine you're building a video streaming platform where a video’s buffer is represented as a binary array of 0s (lag) and 1s (smooth playback). You can flip at most one 0 to 1. Your task is to find the maximum number of consecutive 1s possible in the array after flipping at most one 0.

🧪 Sample Input:


const stream = [1, 0, 1, 1, 0, 1, 1, 1];

// Output:
5

💡 Explanation:

  • By flipping the second 0 (index 4), we get a longer stretch of 5 continuous 1s.

🔸 Real-World Link:

  • Buffering logic, error correction in transmission, or even behavioral analysis in app usage—all apply similar logic to improve smooth sequences with minimal correction.

Day 5 JavaScript Challenge — “Longest Word Extractor”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given a sentence in the form of a string, write a function to find and return the longest word in that sentence.

🧪 Example Input: 

const input = "JavaScript is simple but powerful";

🔸 Your Task to Perform

Break the string into words.

Loop through the words and find the one with the maximum length.

Return that longest word.

🧪 Example Output: 

"JavaScript" // because it is the longest word

🔸 Bonus Tip for You

Do not use built-in methods like `.sort()` or `.reduce()`.

Use basic loops and `.split()` only.

Handle punctuation like `.` or `,` (if present) — remove them before comparison.

Day 79 JavaScript Challenge — “Filter Eligible Candidates”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You are building a dashboard for an HR portal that filters out eligible candidates for a coding bootcamp. Each applicant is represented as an object with fields like `name`, `age`, `experience`, and `languages`.

Your job is to:

  • Only allow candidates aged between 18–30
  • Must have at least 1 year of experience
  • Must know either "JavaScript" or "Python"

🧪 Sample Input:


const applicants = [
  { name: "Ankit", age: 24, experience: 2, languages: ["C++", "Python"] },
  { name: "Sara", age: 31, experience: 4, languages: ["JavaScript"] },
  { name: "Neha", age: 27, experience: 0, languages: ["Python"] },
  { name: "Rahul", age: 22, experience: 1, languages: ["Java", "JavaScript"] },
  { name: "Priya", age: 29, experience: 3, languages: ["Go"] },
];

✅ Expected Output:


[
  { name: "Ankit", age: 24, experience: 2, languages: ["C++", "Python"] },
  { name: "Rahul", age: 22, experience: 1, languages: ["Java", "JavaScript"] }
]

🔸 Bonus Tip

  • Use filter() method to apply multiple conditions
  • Consider case-insensitive match using .toLowerCase() when checking for languages

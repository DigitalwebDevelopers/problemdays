Day 61 JavaScript Challenge — “Two Numbers Sum to Target”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You have an array of integers and a target sum. Find the indices of two distinct numbers whose sum equals the target. This is a classic interview-style problem that introduces you to the **Two Sum** pattern :contentReference[oaicite:3]{index=3}.

🧪 Example Input:

const nums = [2, 7, 11, 15];
const target = 9;

🔸 Your Task to Perform

  • Use a single loop + an object (or `Map`) to record seen numbers.
  • For each number, check if (target – current) exists in the record; if yes, return indices.
  • Return as soon as you find a valid pair.

🧪 Expected Output:

// Output: [0, 1]

🔸 Bonus Tip for You

  • Don’t use nested loops — focus on O(n) solution with hashing.
  • Handle negative numbers and zeros too.

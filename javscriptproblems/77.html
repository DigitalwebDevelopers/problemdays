Day 77 JavaScript Challenge — “Newsletter Subscriber Cleaner”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Suppose you’re managing a newsletter platform and you've received a messy array of subscriber emails. Some are repeated, some invalid, and some are just blank or empty.

You need to clean the list and provide a report showing:

  • Number of valid emails (no duplicates)
  • Number of invalid email formats (e.g., "code@@.com")
  • Number of blank or whitespace-only entries
  • Final array of unique valid emails

🧪 Sample Input:


const subscribers = [
  "john@example.com", "  ", "john@example.com",
  "mary@email.com", "noemail",
  "code@@dev.com", "admin@site.org"
];

🧪 Output:


{
  valid: 3, invalid: 2, empty: 1,
  cleanedEmails: [
    "john@example.com",
    "mary@email.com", "admin@site.org"
  ]
}

🔸 Bonus Tip

  • A simple regex for email format check is /^[^\s@]+@[^\s@]+\.[^\s@]+$/
  • Use Set to remove duplicates efficiently.
  • Trim whitespace and check if a string is empty using str.trim() === ""

Day 43 JavaScript Challenge — “Index Mapping by Value”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of values, return an object where each key is the value, and the corresponding value is an array of all the indexes at which that value occurs.

🧪 Example Input:

const input = ['a', 'b', 'a', 'c', 'b', 'a'];

🔸 Your Task to Perform

Loop through the array. For each value, store its index inside an array under its own key in an object.

If a value occurs multiple times, collect all its index positions.

🧪 Example Output:

{
  a: [0, 2, 5],
  b: [1, 4],
  c: [3]
}

🔸 Bonus Tip for You

Don’t use filter(). Use a simple loop and a plain object to build the mapping.

Try handling inputs with numbers, strings, and mixed values as keys.

Day 34 JavaScript Challenge — “Palindrome Detector in Array”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

Given an array of strings, return the first string that is a palindrome. If no palindrome exists, return "Not Found".

🧪 Example Input: 

const words = ["level", "world", "deed", "test"];

🔸 Your Task to Perform

  • Loop through the array.
  • Check for each word if it reads the same backward.
  • Ignore case sensitivity while checking.

🧪 Expected Output: 

"level" // First palindrome found

🔸 Bonus Tip for You

Don’t use built-in `.reverse()` — reverse using a loop or custom method for deeper understanding.

Try ignoring spaces or special characters (extra challenge).

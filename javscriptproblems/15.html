Day 15 JavaScript Challenge — “Detect Repeating Patterns”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You're given an array of strings. You have to detect if any **word pattern** (substrings) repeats **at least 3 times consecutively** in the array.

🧪 Example Input:

const arr = ["go", "go", "go", "stop", "go"];

🔸 Your Task to Perform

Loop through the array and check if any string appears 3 or more times **back to back**.

If such a pattern is found, return true. Otherwise return false.

🧪 Example Output:

true // "go" is repeated 3 times continuously

🔸 Bonus Tip for You

Don't use array methods like filter(), reduce() or groupBy(). Use only simple loops and counter logic.

Try with lowercase/uppercase mix — make solution case-sensitive.

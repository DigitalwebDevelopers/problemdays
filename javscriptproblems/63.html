Day 63 JavaScript Challenge — “Item Frequency Tracker”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re working on a retail app where a function receives an array of sold product names throughout the day. You need to return an **object** representing how many times each product was sold.

🧪 Example Input:

const soldItems = [
  "pen", "notebook", "pen", "eraser", "pencil", "notebook", "pen"
];

🔸 Your Task to Perform

  • Loop through the array and count how many times each item appears.
  • Return an object where keys are product names and values are their frequencies.

🧪 Expected Output:

{
  pen: 3, notebook: 2,
  eraser: 1, pencil: 1
}

🔸 Bonus Tip for You

  • Try using `.reduce()` as an alternative to the `for...of` loop.
  • Consider converting the object back into a sorted array for reports.

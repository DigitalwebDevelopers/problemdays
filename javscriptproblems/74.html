Day 74 JavaScript Challenge — “Budget Category Analyzer”

Taught By Code with Deepanshu Knox

🔍 Problem Of the Day —

You’re building a finance tracker. A user inputs their monthly expenses categorized under types like "Groceries", "Entertainment", "Bills", etc.

Your job is to create a function that: calculates the total for each category and then identifies the category where most money was spent.

🧪 Sample Input:

const expenses = [
  { category: "Groceries", amount: 1200 },
  { category: "Entertainment", amount: 600 },
  { category: "Groceries", amount: 800 },
  { category: "Bills", amount: 1500 },
  { category: "Entertainment", amount: 900 }
];

🔸 Expected Output:

{
  totals: {
    Groceries: 2000,
    Entertainment: 1500,
    Bills: 1500
  },
  highest: "Groceries"
}

🔸 Bonus Tip

  • Use a JS object to aggregate totals per category efficiently.
  • Use `Object.entries()` to determine the category with max spending.
